# 홈 CCTV 특화 객체 인식 모델 개발 계획

## 1. 프로젝트 개요
- 목적: 홈 CCTV 환경에 최적화된 객체 인식 모델 개발
- 적용 분야: 가정 내 보안, 어린이/반려동물 모니터링, 비정상 행동 감지
- 목표 성능: mAP(mean Average Precision) 0.85 이상, 실내 환경에서의 정확도 우선

## 2. 타겟 객체 정의
- 핵심 인식 대상: 사람(가족/침입자), 반려동물(개/고양이), 위험 물체(칼/총기), 화재/연기
- 보조 인식 대상: 창문/문 열림 상태, 택배 상자, 귀중품(지갑/전자기기), 어린이 행동
- 불필요 객체 제외: 자동차, 자전거, 가로등 등 실외 전용 객체

## 3. 데이터 준비
- 데이터 획득 방법:
  1) 공개 데이터셋(COCO, Open Images)에서 필요 클래스만 선별
  2) 직접 가정 환경 데이터 수집 및 라벨링
- 데이터 필터링: 불필요 클래스 제거를 위한 스크립트 개발
- 데이터 증강: 실내 조명 조건, 시점 변화 등 홈 환경 특화 증강

## 4. 모델 아키텍처 선택
- 기본 모델: YOLOv5 계열 유지 (선행 경험 활용)
- 백본 최적화: 실내 소형 객체 인식에 적합한 백본 선택
- 경량화: 저사양 CCTV 하드웨어에서도 실시간 처리 가능하도록 설계

## 5. 모델 학습
- 학습 데이터: 필터링된 공개 데이터셋 + 홈 환경 커스텀 데이터
- 학습 전략: 
  1) 선별된 공개 데이터로 사전 학습
  2) 홈 환경 데이터로 미세 조정(fine-tuning)
- 하이퍼파라미터: 작은 객체 감지 강화를 위한 최적화

## 6. 모델 평가 및 개선
- 평가 환경: 실제 가정 내 CCTV 설치 환경과 유사한 조건
- 특화 테스트: 낮/밤 조명 변화, 가려짐(occlusion), 다양한 실내 각도
- 오류 분석: 특히 오인식(false positive)을 줄이는 방향으로 개선
- 임계값 조정: 홈 환경에 맞는 신뢰도 임계값 최적화

## 7. 모델 최적화
- 모델 경량화: 양자화, 가지치기로 Edge 디바이스 최적화
- 추론 최적화: ONNX, TensorRT 변환으로 처리 속도 개선
- 전력 효율성: 배터리 기반 CCTV를 위한 저전력 모드 개발

## 8. 배포 계획
- 타겟 플랫폼: 홈 CCTV 시스템, 스마트홈 허브
- 알림 시스템: 중요 객체 감지 시 즉시 알림 기능 구현
- 사생활 보호: 가족 구성원 식별 및 프라이버시 설정 기능

## 9. 일정 계획
- 1단계: 데이터 선별 및 필터링 스크립트 개발 (1주)
- 2단계: 홈 환경 데이터 수집 및 공개 데이터 필터링 (2주)
- 3단계: 기본 모델 학습 및 홈 환경 미세 조정 (2주)
- 4단계: 실제 환경 테스트 및 오류 분석 (2주)
- 5단계: 모델 최적화 및 배포 준비 (1주)

## 10. 필요 자원
- 하드웨어: 학습용 GPU 서버, 테스트용 CCTV 장비
- 소프트웨어: PyTorch, OpenCV, 라벨링 도구
- 데이터: 실내 환경 이미지/영상, 특히 다양한 조명 조건
